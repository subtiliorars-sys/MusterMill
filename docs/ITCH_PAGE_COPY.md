## Full description

**MusterMill** is a browser idle game spun off from the **Mustermate Network** dating-app parody universe. You are not swiping. You are **mustering**.

### Core loop

1. **Legacy Muster** — select two adult Specialists and assign them to Heritage Quarters.  
2. **Hatch recruits** — babies inherit MOS-flavored traits (11B Bloodline, 68W Heritage, Joint Task Force, …).  
3. **Boot camp timers** — Recruit → Private → Specialist.  
4. **KP Duty** — send adults on idle missions to earn **Muster Slips**.  
5. **Branch skins** — free Army/OCP palette; premium branch packs mirror HooahHook, FleetMate, SemperSwipe, and the rest.

### Controls

- Click soldier cards to select (two Specialists to pair).  
- **Legacy Muster** — start pairing timer.  
- **Send to KP** — idle currency while you tab away.  
- **Reinforcement** — spend 40 slips for a ready Specialist.  
- **Branch skin** chips at top — swap battalion colors and mascot emoji.

### Branch skin toggle (what it is)

Same game engine, different **Mustermate branch wrapper**: colors, mascot (🪖 ⚓ 🦅 ✈️ 🛰️ 🚤 🎖️), and subtitle. Free tier uses OCP / Army. Other branches are labeled **Pack** — unlocked in this itch demo via **Preview all branches**.

### What this is NOT

- Not affiliated with the U.S. Department of Defense.  
- Not a dating sim or visual novel — pairing is a **production mechanic**.  
- Fictional chibi soldiers only.
